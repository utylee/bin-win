Boston Open Source Science Laboratory or BOSSLab is a labspace in Somerville, MA, which is part of the DIYbio and biohacking movement.

Major projects[edit]

  • Lightbulb PCR, which aims to build a cheap but functional PCR thermal cycler using PVC tubing, a incandescent light bulb and a computer fan run with an open source microcontroller.
  • BlueGene- Indigo Biosynthesis In E. Coli

Ethical concerns[edit]

The institute of Ethics and Emerging Technologies covered the ethics of BOSSLab and other DIYbio groups.[1]
