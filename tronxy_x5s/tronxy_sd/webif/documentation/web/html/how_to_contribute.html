Get involved
Grab your Starter kit and learn how you can contribute to MediaWiki and the free culture community.
Web APIsAPI:Web APIs hub
The content of all Wikimedia projects is released under free licenses. Write code to access, remix and grow this immense pool of free knowledge using the robust action API, available on all MediaWiki wikis, and other APIs for content and Wikidata. Other open data sources, including XML and SQL dumps, are also available.
DevelopmentDeveloper hub
Patches welcome! You can improve MediaWiki core and its extensions, its appearance via skins, user customizations like scripts and gadgets, or templates to process content.

Create bots to process content and host your tools on Labs. Hack on mobile apps or on desktop applications like Huggle and AWB. Or help Site Operations maintain the Wikimedia infrastructure.

The code is all free and open source. Choose a coding project, big or small! The main languages are PHP, JavaScript, HTML and CSS, as well as Lua for some templates and extensions.
TestingQA
Report your first bug! Also help improve the quality of our projects through manual testing, automated browser testing, continuous integration and bug management.
Ambassadors
As a Tech ambassador, help other Wikimedians with technical issues, relay Tech news to inform users about what is going to impact them, and join the ambassadors list to act as a bridge between developers and your local wiki.
DocumentationDocumentation
English writers can improve the MediaWiki documentation, other essential support pages and, in fact, any page of this website.
TranslationLocalization
If you are fluent in a language other than English you can join the effort by translating this website and the MediaWiki software.
Support
Help users and developers looking for answers at the support desk or the MediaWiki communication and social media channels.
Product developmentProduct development
What do users need? What features should we develop? What bugs should we prioritize? Help planning the right tasks with designers and developers.
DesignDesign
Help apply the Wikimedia design principles in projects looking for UX feedback, small improvements, multilingual and mobile interfaces.
Meet upCommunication#com-landing
Meet the community in cyberspace or in person.
