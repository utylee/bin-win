iMX6 DualLite COM Board

Embedded Artists' iMX6 DualLite COM Board is equipped with NXP's Cortex-A9 based i.MX6 DualLite processor suitable for a wide range of industrial applications that requires advanced communication, real-time control and high quality graphic displays.

The i.MX 6DualLite is a dual core, high performance and energy efficient processor operting at speeds up to 1 GHz. The processor is ideal for running an OS like Linux.

Use the iMX6 DualLite Developer's Kit to get up-and-running quickly with the board.

Specification

Processor NXP i.MX 6DualLite with dual 1 GHz/800 MHz ARM Cortex-A9 cores
Memory 1 GByte DDR3L 800 MT/s, 64-bit databus
4 GByte eMMC NAND Flash for OS and bootloader
Graphics output

Up to two display interfaces can be active simultaneous;
• Dual LVDS: 18/24 bit, up to 85 Mpixels/sec, for example WXGA (1366 x 768 px) at 60 Hz
• Parallel RGB: 24-bit up to WXGA (1366 x 768 px) at 60 Hz
• MIPI-DSI: 2 lanes
• HDMI 1.4 port: up to 1920 x 1080 px

Graphics Engines: GPU 3D/2D (GC880/GC320) supporting OpenGL ES 1.1/2.0, OpenVG 1.1 APIs
HW video decode: 1080p30h + D1
HW video encode: 1080p30h H.264 BP / Dual 720p encode 
Graphics input CMOS sensor interface (camera), parallel interface,
and serial interface: MIPI-CSI2, 2 lanes
Ethernet 10/100/1000 Mbps Gigabit Ethernet interface with on-board Ethernet-PHY
I/O i.MX 6DualLite peripherals (see i.MX 6DualLite datasheet for details about peripherals):
• PCIe; 1x lane, v2.0
• USB; 1x USB 2.0 Host, 1x USB 2.0 OTG
• Audio; 3x SSI, ESAI
• SPDIF; 1 channel
• SDIO; 3x MMC/SD/SDIO
• Serial Port; 5x UART
• SPI; 4x eCSPI
• CAN; 2x CAN bus 2.0B
• I2C; 3x channels
• PWM; 4x channels
See iMX6 DualLite COM Board Datasheet for details about available pins/peripherals.
Mechanical 82 x 50 mm (W x D), SMARC form factor but different pinning for better carrier board routing
Power +3.3V power supply voltage
Power consumption: TBD
Connectors 314 pos MXM3 edge connector, 0.5mm pitch
10 pos, 0.5mm pitch FPC for JTAG
Environment Operating Temperature: 0 - 60 degrees Celsius (standard) or -40 to +85 degrees Celsius (on request)
Operating Humidity: 5 - 90% relative humidity, non-condensing
Other • i.MX 6DualLite on-chip RTC
• On-board watchdog functionality
• Integrated power managment with PMIC (PF0100) supporting DVFS techniques for low power modes
Product Classification
and Status		 Integration board; In Development/Sampling
