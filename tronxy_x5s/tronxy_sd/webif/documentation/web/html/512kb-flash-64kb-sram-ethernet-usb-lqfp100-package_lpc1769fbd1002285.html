概述

LPC1769是一种适合嵌入式应用的Cortex-M3微控制器，具有较高的整合度和低功耗，运行频率为120 MHz。功能包括512 kB闪存、64 kB数据存储器、以太网MAC、USB 设备/主机/OTG、8通道DMA控制器、4个UART、2个CAN通道、3个SSP/SPI、3个I2C、I2S、8通道12位ADC、10位DAC、电机控制PWM、正交编码器接口、4个通用定时器、6输出通用PWM、带独立电池供电的超低功耗实时时钟以及最多70个通用I/O引脚。LPC1769的引脚兼容100引脚的LPC2368 ARM7 MCU

特性

  • ARM Cortex-M3处理器，工作频率高达120 MHz
  • ARM Cortex-M3内置可嵌套矢量中断控制器(NVIC)
  • 高达512 KB的片内闪存编程存储器
  • 高达64 kB片内SRAM
  • 在系统编程(ISP)和在应用编程(IAP)
  • 8通道通用DMA控制器(GPDMA)
  • 以太网MAC，带RMII接口和专用DMA控制器
  • USB 2.0全速设备/主机/OTG控制器
  • 4个带小数波特率生成器、内部FIFO和DMA支持的UART
  • 双通道CAN 2.0B控制器
  • 带同步、串行、全双工通信的SPI控制器
  • 2个SSP控制器，带FIFO和多协议功能
  • 3个增强型I2C总线接口
  • I2S（Inter-IC音频）接口
  • 70个通用I/O (GPIO)引脚，带可配置的上拉/下拉电阻
  • 12位/8通道模拟/数字转换器(ADC)，转换速率高达200 kHz
  • 10位数字/模拟转换器(DAC)，带专用转换定时器和DMA
  • 4个通用定时器/计数器
  • 一个电机控制PWM，支持三相电机控制
  • 正交编码器接口，可监控一个外部正交编码器
  • 一个标准PWM/定时器模块，带外部计数输入
  • 低功耗RTC，带独立的电源域和专用振荡器
  • 看门狗定时器(WDT)
  • ARM Cortex-M3系统节拍定时器，包括一个外部时钟输入选项
  • 重复中断定时器提供可编程的和重复定时中断
  • 为进一步节电，每个外设都有其自己的时钟分频器
  • 与现有工具兼容的标准JTAG测试/调试接口
  • 集成式PMU（电源管理单元）
  • 4种低功耗模式：睡眠模式、深度睡眠模式、掉电模式和深度掉电模式
  • 3.3 V单电源（2.4 V至3.6 V）
  • 四个可配置为边沿/电平触发的外部中断输入
  • 非屏蔽中断(NMI)输入
  • 唤醒中断控制器(WIC)
  • 处理器通过任何中断从省电模式唤醒
  • 以可触发中断和强制复位的独立阈值进行欠压检测
  • 上电复位(POR)
  • 工作范围从1 MHz到25 MHz的晶体振荡器
  • 校准到1％精度的4 MHz内部RC振荡器
  • 具有不同安全级别的代码读取保护(CRP)
  • 可用作芯片识别的唯一序列号
