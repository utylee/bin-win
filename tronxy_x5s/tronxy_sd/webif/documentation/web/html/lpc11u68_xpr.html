LPC11U68 LPCXpresso Board

The LPC11U68 LPCXpresso board with NXP's ARM Cortex-M0+ microcontroller has been designed to make it as easy as possible to get started with Cortex-M0+.

An LPCXpresso board consists of a target side (LPC11U68, in this case) and a build-in LPC-Link2 debug probe, which is compatible with the free LPCXpresso IDE out-of-the-box, and with other toolchains via optional ARM CMSIS-DAP and Segger J-Link firmware.

The LPC11U68 has 36 kB SRAM, 256 kB Flash, 4 kB E2PROM, 5xUART, 2xI2C, 2xSPI/SSP, PWM, USB 2.0 Device, RTC, etc.

LPCXpresso V2

Since the introduction of the first variant in 2009, the original LPCXpresso board has reenergised the whole MCU evaluation board market. Embedded Artists and NXP are now proud to introduce the new generation of LPCXpresso V2 boards which build upon the original design, providing many new and exciting features.

LPCOpen

The LPCOpen Platform allows users to quickly and easily utilize NXP's extensive array of microcontroller software libraries to create and develop multifunctional products. The platform uses common APIs that work equally across all microcontroller families and can be built with Keil, IAR, and LPCXpresso (Eclipse-based tool) tool chains.

mbed SDK

The LPC11U68 LPCXpresso will soon be supported by the mbed framework.

Specification

LPC11U68 LPCXpresso Board

Processor NXP's Cortex-M0+ LPC11U68 microcontroller in LQFP100 package
Flash Memory 256 kB
Data Memory 36 kB
E2PROM 4 kB
Clock Crystals 12.000 MHz and 32.768 kHz crystals for CPU
Dimensions 55 x 83 mm
Power
  • 3.3V external powering,
  • 5V external powering,
  • LPC11U68 USB interface, or
  • LPC-Link2 USB interface (debug interface)
Connectors
  • ArduinoTM connectors compatible with the 'Arduino UNO' platform
  • All relevant LPC11U68 pins available on 'standard LPCXpresso/mbed' expansion connector (2x27 pin rows, 100 mil pitch, 900 mil between rows)
  • Micro-B USB connector for LPC11U68 USB interface
  • SWD interface (2x5 pos, 50 mil pitch) to LPC11U68 (built-in debug probe can be overridden by externally connected debugger)
Other
  • Built-in LPC-Link 2 compatible debug proble, based on NXP LPC4300 MCU.
  • RGB-LED
  • ISP-enable and WAKEUP push-buttons
  • Reset push-button and LED
  • Current consumption can be measured of all target side circuitry
