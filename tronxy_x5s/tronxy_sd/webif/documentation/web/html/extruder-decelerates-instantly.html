Extruder Decelerates Instantly
kbranch 30 Apr 2016 01:11

I'm trying to debug an issue I'm having where my extruder makes worrying noises during fast retracts, and I noticed that the extruder stepper seems to ignore acceleration when stopping - it slams to a complete stop instantly. I'm not positive that this is the source of my noise yet, but it seems a bit aggressive once you add the inertia from the second gear and hobbed bolt.

Is this expected? Is there a way to have it decelerate as slowly as it accelerates?

This is the result of G1 E+300 F4000 with extruder.hotend.acceleration set to 5: youtu.be (slash) OamXJfZaLJE (forgive the mangled URL, it won't let me post links yet)

I get the same behavior with the latest stable build and the latest nightly (edge-88e84b2). Here's my full config:

Hey

That is expected, deceleration has never been implemented on extruder, and from the user feedback I hear, for retractation that is actually a good thing.
You likely want a much much higher acceleration value for your extruder anyway, as much as you can go to without it stalling.

Cheers.
