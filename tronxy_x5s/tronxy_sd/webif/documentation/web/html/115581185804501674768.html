to improve the resolution of the OWI sensor for 1.75mm filament, you need to turn the sensor 45 degrees so that it measures the filament on a diagonal. You then get a ~40% improvement in resolution. Of course, this makes it a bit slower at detecting rapid filament diameter changes.

Both the toy builder sensor and the OWI sensor can detect "out of filament" but there is no current way for the OWI to detect stopped filament, which is the toy builder's primary purpose in my printer.﻿

Here's a video of my machine with an AMP 9 pin mil spec connector (I found in a box at the makerspace- probably about a $300 connector!): https://vimeo.com/114485288 Here's a video when I was using power-pole connectors:  https://vimeo.com/78444620 Here's one with the header pins and a flex ribbon cable plugged in- the current arrangement that I don't intend to change: https://vimeo.com/137001912

The AMP connector was better than the power-pole connectors because it was easy to plug and unplug compared to power poles, and about the same size, but the header pins seems best to me now.  I use header pins on the circuit board to plug in the extruder motor, hot-end fan, thermistor, and print cooling fan, and there are a few extras for not yet installed LEDs to light up the print.  The heater is connected via a screw terminal block.  The extruder carriage circuit board is here: http://cdn.instructables.com/FYR/WWLN/I81GDZNH/FYRWWLNI81GDZNH.LARGE.jpg

Note- the ribbon cable I used is a flex-ribbon, not the standard ribbon cable.  You might need more pins to share the current for the heater with a standard ribbon cable.﻿
