Product downloads

Click on the operating system that you wish to run LPCXpresso IDE on to download the LPCXpresso IDE installer...

  • Windows download 
    • Windows Vista or later is required. 32-bit and 64-bit systems are supported
  • Linux download 
    • Supported distributions are listed in the User Guide. 32-bit and 64-bit systems are supported
  • Mac OS X download
    • Mac OS X Mountain Lion (10.8.5) or later is required

Once downloaded and installed, you can activate LPCXpresso IDE (Free Edition) or purchase LPCXpresso IDE (Pro Edition) directly from the NXP LPCXpresso IDE webstore and then activate LPCXpresso IDE (Pro Edition).
