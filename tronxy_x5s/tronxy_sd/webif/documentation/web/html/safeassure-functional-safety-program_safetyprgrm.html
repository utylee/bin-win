SafeAssure Functional Safety Program

A simpler way to comply with standards for functional safety at the system level.

These SafeAssure solutions reduce the time it takes to develop safety systems that comply with the International Standards Organization (ISO) 26262 and International Electrotechnical Commission (IEC) 61508 standard.

SafeAssure Program Features
  • Select MCUs, mixed-signal MCUs, processors, analog and power management ICs and sensors
  • Hardware safety concepts focused on detecting and mitigating random hardware failures using built-in safety features, including self-testing, monitoring and hardware-based redundancy
  • Software that works seamlessly with hardware for system-level functional safety goals
  • Support, including:
    • Customer-specific training
    • System design reviews regarding functional safety architecture
    • Safety documents
    • Technical support
    • SafeAssure product-specific safety documents, upon request

This program supports the most stringent Safety Integrity Levels (SILs) and Automotive Safety Integrity Levels (ASILs) to help developers get to system compliance more easily.
