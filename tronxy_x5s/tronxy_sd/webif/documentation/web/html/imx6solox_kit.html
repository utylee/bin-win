iMX6 SoloX Developer's Kit

Embedded Artists' iMX6SoloX Developer's Kit lets you get up-and-running quickly with the iMX6SoloX COM Board. Besides the iMX6SoloX COM Board the kit contains the COM Carrier Board, which is a carrier board designed for direct integration. All relevant interfaces are available - either for prototyping/evaluation or for direct integration into your product!

The on-board mini-PCIe, SDIO and XBee™ interfaces makes it very simple to add RF solutions.

See Display Solutions for COM boards for details about how easy it is to add a display to your project!

See our EACOM Selection Guide if you need help finding a suitable EACOM board.

COM Carrier Board specification

The information listed below for the COM Carrier Board is valid when the iMX6 SoloX COM board is used. The COM Carrier Board has more interface connectors than listed below, but these interfaces are not supported by the i.MX 6SoloX processor.

Connectors and
Interfaces		 • MXM3, 314-pos connector with standoffs for COM boards
• Dual 10/100/1000 Mbps Ethernet RJ45 connectors
• USB OTG interface with ESD protection
• Dual USB Host interfaces (via USB Hub) with ESD protection
• uSD/MMC connector (MMC is in parallel with uSD, so either one can be used but not both)
• Dual CAN transceivers with ESD protection and optional termination, accessible via Molex Micro-Fit 3.0 connectors
• Internal UART connectors (compatible with FTDI cables)
• XBee™ compatible interface connector
• Mini PCIe connector for half and full-size cards, including SIM card holder and USB interface
• FPC connectors for parallel camera interface
• LVDS connector
• Parallel 18-bit RGB interface for 7" LCD solutions 
• Expansion connectors for many COM Board signals
• Audio codec with line out 3.5mm audio jack connector (with internal connector for microphone and line input and headphone output)
Power • 12V (+-30%) supply voltage, either via internal ATX connector or via external Molex Micro-Fit 3.0 connector
• Reverse polarity protection
• Internal 3.3V/4A and 5V/3A (and 3.3V/3A for PCIe) DC/DC converters with high efficiency
• Coin cell battery holder (CR1220 size) for COM board RTC (note: coin cell not included)
Dimensions • 165 x 104 mm, same size as typical 7 inch LCDs.
• Five M3 holes for mounting and grounding
Other • On/Off and Reset pushbuttons
• Input current measurement
