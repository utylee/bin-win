Guidelines
Y
  • DO place this template on articles for products that are specified to be for Linux by the manufacturer and are available bundled with Linux
N
  • DO NOT place this template on articles for products that can run Linux but are not shipped with it

Usage[edit]

How to manage this template's initial visibility
To manage this template's visibility when it first appears, add the parameter:

|state=collapsed to show the template in its collapsed state, i.e. hidden apart from its titlebar – e.g. {{Linux devices |state=collapsed}}
|state=expanded to show the template in its expanded state, i.e. fully visible – e.g. {{Linux devices |state=expanded}}
|state=autocollapse to show the template in its collapsed state but only if there is another template of the same type on the page – e.g. {{Linux devices |state=autocollapse}}

Unless set otherwise (see the |state= parameter in the template's code), the template's default state is autocollapse.
